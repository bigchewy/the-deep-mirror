## 3. ON-PAGE STRATEGY

### HEADING HIERARCHY TEMPLATE
```
H1: Primary Keyword + Value Proposition
H2: Core Topic Sections (include semantic keywords)
H3: Detailed Subtopics (natural keyword variations)
H4: Technical details, examples, steps
```

**Example Structure:**
```
H1: AI Enneagram Assessment vs Traditional Test: Accuracy Comparison 2024
H2: Why Traditional Enneagram Tests Fail
H3: Social Desirability Bias in Personality Assessments
H3: Limited Question Logic Problems
H2: How Conversational AI Solves Assessment Problems  
H3: Dynamic Interviewing Capabilities
H3: Context Integration Features
H2: Clinical Grade Accuracy: The Evidence
```

### KEYWORD PLACEMENT STRATEGY
- **Primary keyword:** H1, first 150 words, at least 2 H2s, meta title
- **Supporting keywords:** H2/H3 headings, naturally throughout body
- **Semantic keywords:** Scattered naturally, focus on user intent
- **LSI terms:** Questions, problems, solutions related to main topic

### META DESCRIPTION APPROACH
Template: "Problem/Question + Solution + Unique Value + CTA"

Examples:
- "Why traditional enneagram tests mistype executives. How AI-powered conversational assessment delivers clinical-grade accuracy. Compare methods and try advanced typing."
- "Social desirability bias ruins 73% of personality test results. Discover how conversational AI eliminates response manipulation for accurate enneagram typing."
